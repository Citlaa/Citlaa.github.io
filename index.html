Política de Privacidad para el Proyecto "Nueva Comunicación"

1. Información General

El proyecto "Nueva Comunicación" es una investigación independiente cuyo objetivo principal es evaluar el impacto de campañas a través del análisis de datos públicos de páginas de Facebook, utilizando técnicas de data mining. Esta política de privacidad describe cómo se recopilarán, utilizarán, protegerán y almacenarán los datos obtenidos.

2. Datos Recopilados

En el marco de este proyecto, se recopilarán datos públicos de páginas de Facebook, incluyendo pero no limitándose a:

  • Publicaciones
  • Comentarios
  • Reacciones (Me gusta, Me encanta, etc.)
  • Compartidos de contenido público

Nota: No se recopilarán datos de usuarios individuales, solo de contenido accesible públicamente en las páginas de Facebook.

3. Uso de los Datos

Los datos recopilados serán utilizados únicamente con fines de investigación para evaluar el impacto de las campañas en las páginas públicas de Facebook.

  • El análisis de los datos se utilizará para identificar patrones y tendencias en la interacción del público con las campañas.
  • No se compartirán los datos con terceros fuera del equipo de investigación.
  • Los resultados del análisis serán publicados en formato agregado y anónimo, sin revelar ninguna información personal o identificable.

4. Seguridad y Protección de Datos

El equipo de investigación implementará medidas de seguridad adecuadas para proteger los datos recopilados, incluyendo:

  • Cifrado de datos: Los datos estarán protegidos mediante cifrado durante su recopilación y almacenamiento.
  • Acceso restringido: Solo el equipo autorizado tendrá acceso a los datos recopilados.
  • Almacenamiento seguro: Los datos serán almacenados en servidores seguros durante el análisis.

Este proyecto cumplirá con las normativas de protección de datos aplicables, como el Reglamento General de Protección de Datos (GDPR) en Europa, y otras normativas locales según corresponda.
